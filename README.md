An AI-based decision support system that analyzes plant infection severity and environmental conditions to recommend appropriate pesticide application.

## Features
- Leaf infection severity detection
- Real-time weather analysis
- Intelligent spraying decision
- Spray duration simulation
- Plant health visualization

## How It Works
1. Upload plant leaf image
2. System calculates infection percentage
3. Fetches environmental conditions
4. Determines spraying necessity
5. Simulates pesticide application

## Installation

Clone repository:
git clone <repo-link>

Go to folder:
cd intelligent-pesticide-system

Install dependencies:
pip install -r requirements.txt

Run project:
streamlit run app.py

## Objective
Reduce unnecessary pesticide usage and promote precision agriculture.
